## Built-in Features

Everything you need to launch and scale your SaaS:

| Feature | Description |
|---------|-------------|
| **Authentication** | Email/password, social login, 2FA, email verification |
| **Multi-tenancy** | Team workspaces with role-based access control |
| **Payments** | Stripe integration with subscriptions and one-time payments |
| **Notifications** | Real-time in-app notifications |
| **Email System** | Transactional emails with templates |
| **CI/CD Pipeline** | Automated deployments to AWS |
| **Admin Panel** | Django admin with custom actions |
| **GraphQL API** | Type-safe API with code generation |
