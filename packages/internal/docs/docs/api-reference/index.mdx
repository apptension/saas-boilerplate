## Quick Reference

### Common Commands

| Package | Command | Description |
|---------|---------|-------------|
| **All** | `pnpm saas up` | Start backend services and webapp dev server |
| **All** | `pnpm saas down` | Stop backend and frontend services |
| **Webapp** | `pnpm nx start webapp` | Start development server |
| **Webapp** | `pnpm nx test webapp` | Run frontend tests |
| **Webapp** | `pnpm nx run webapp:type-check` | TypeScript type checking |
| **Backend** | `pnpm saas backend shell` | Django shell access |
| **Backend** | `pnpm saas backend test` | Run backend tests |
| **Backend** | `pnpm saas backend makemigrations` | Create database migrations |
| **Backend** | `pnpm saas backend migrate` | Apply database migrations |
| **API Client** | `pnpm nx run webapp:graphql:generate-types` | Generate GraphQL types |
| **API Client** | `pnpm nx run webapp:graphql:download-schema` | Download GraphQL schema |
| **Docs** | `pnpm nx start docs` | Start documentation server |
