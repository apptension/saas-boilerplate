SaaS Boilerplate is a fully functional codebase that allows you to radically decrease the time your team spends on developing a SaaS application. Your development team can re-use common components and focus on the core of your business.

**What's included:**

- **Infrastructure** — AWS-based deployment with [CI/CD pipelines](/introduction/features/cicd) built-in
- **[Payments & Subscriptions](/introduction/features/payments)** — Stripe integration for billing and subscriptions
- **[Multi-tenancy](/introduction/features/multi-tenancy)** — Team workspaces with role-based access
- **[Authentication](/introduction/features/auth)** — Login, signup, 2FA, social auth, and more
- **[CMS Integration](/introduction/features/cms)** — Contentful for content management
- **[CRUD Generators](/introduction/features/crud)** — Quickly scaffold new features
- **[And much more...](/introduction/features)** — Emails, notifications, admin panel, and more
