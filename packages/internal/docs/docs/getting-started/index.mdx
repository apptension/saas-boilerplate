## Quick Start (TL;DR)

For experienced developers who want to jump right in:

```bash
# Create a new project
pnpm create saas-boilerplate my-saas-app
cd my-saas-app

# Start everything
pnpm saas up
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

:::tip What You Get
A fully functional SaaS application with authentication, payments, multi-tenancy, and more — all running locally.
:::

## Prerequisites

Make sure you have these installed before starting:

| Tool        | Version | Check Command      | Install                                          |
| ----------- | ------- | ------------------ | ------------------------------------------------ |
| **Node.js** | 20+     | `node -v`          | [nodejs.org](https://nodejs.org)                 |
| **pnpm**    | 9+      | `pnpm --version`   | [pnpm.io](https://pnpm.io/installation)          |
| **Docker**  | Latest  | `docker --version` | [docker.com](https://docs.docker.com/get-docker) |

<details>
<summary><strong>Windows Users</strong></summary>

You must install [WSL 2](https://learn.microsoft.com/en-us/windows/wsl/install) (Windows Subsystem for Linux) and run all commands inside WSL.

</details>

<details>
<summary><strong>Optional: Python (for backend development outside Docker)</strong></summary>

If you want IDE autocomplete for Python code or run backend tests locally:

- Python 3.11 (`python3 --version`)
- uv 0.9.29 (`uv --version`)

Most developers don't need this since the backend runs in Docker.

</details>

## What's Running Locally

After setup, you'll have these services available:

| Service         | URL                                                 | Description                |
| --------------- | --------------------------------------------------- | -------------------------- |
| **Web App**     | [localhost:3000](http://localhost:3000)             | Your SaaS frontend         |
| **Backend API** | [localhost:5001](http://localhost:5001)             | Django + GraphQL API       |
| **Admin Panel** | [admin.localhost:5001](http://admin.localhost:5001) | Django admin interface     |
| **Mailcatcher** | [localhost:1080](http://localhost:1080)             | Catches all emails locally |
| **Docs**        | [localhost:3006](http://localhost:3006)             | This documentation         |

---

## Common Commands

After setup, these are the commands you'll use most:

| Command                   | Description                             |
| ------------------------- | --------------------------------------- |
| `pnpm saas up`            | Start all services (backend + frontend) |
| `pnpm saas down`          | Stop all services                       |
| `pnpm nx start webapp`    | Start only the frontend                 |
| `pnpm nx test webapp`     | Run frontend tests                      |
| `pnpm saas backend shell` | Open Django shell                       |

See the [API Reference](/api-reference) for a complete list.
