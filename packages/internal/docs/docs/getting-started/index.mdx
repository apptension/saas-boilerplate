## Quick Start

The fastest way to get started is with our CLI tool:

```bash
pnpm create saas-boilerplate my-saas-project
```

This will:
- Clone the repository
- Install all dependencies
- Set up your environment
- Guide you through initial configuration

:::tip Estimated Time
Most developers are up and running in **under 15 minutes**.
:::

## What You'll Need

Before you begin, make sure you have these prerequisites installed:

| Requirement | Version | Purpose |
|-------------|---------|---------|
| **Node.js** | 18+ LTS | Frontend build & tooling |
| **pnpm** | 9+ | Package management |
| **Docker** | Latest | Backend services |
| **Python** | 3.11 | Backend runtime (optional for local development) |
