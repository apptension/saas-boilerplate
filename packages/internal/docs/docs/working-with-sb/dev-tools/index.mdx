## Plop Generators

Generate boilerplate code quickly:

```bash
cd packages/webapp
pnpm run plop
```

Available generators:
- **Component** — React component with tests and Storybook
- **Route** — New page with routing setup
- **Email** — Email template with Subject and Body

---

## Storybook

Develop and preview UI components in isolation:

```bash
# Start webapp Storybook
pnpm nx storybook webapp

# Start email Storybook
pnpm nx storybook webapp-emails
```

---

## SSH into Containers

Access running containers for debugging:

```bash
# Backend shell
pnpm saas backend shell

# Run Django management commands
pnpm saas backend manage migrate
```
