# Emails & Notifications

<ProjectName /> includes a complete system for transactional emails and in-app notifications.

---

## Email System

Emails are built with React components and rendered to HTML on the backend:

```
React Template → Backend Renders → AWS SES Sends → User Inbox
     │
     └── Storybook Preview
```

**Features:**
- ✅ React-based email templates
- ✅ Internationalization (i18n) support
- ✅ Storybook for local preview
- ✅ Mailcatcher for local testing
- ✅ AWS SES for production delivery

---

## Notification System

In-app notifications are delivered in real-time:

```
Backend Event → GraphQL Subscription → WebSocket → Bell Icon / Toast
```

**Features:**
- ✅ Real-time delivery via WebSocket
- ✅ Persistent storage in database
- ✅ Read/unread status
- ✅ Notification center UI
