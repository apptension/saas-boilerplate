## How Notifications Work

The notification system provides real-time in-app notifications:

```
Backend (Django) → GraphQL Subscription → WebSocket → Frontend (React) → Toast/Bell Icon
```

**Key features:**
- ✅ Real-time delivery via WebSocket
- ✅ Persistent storage in database
- ✅ Read/unread status tracking
- ✅ Notification center UI (bell icon)
- ✅ Toast notifications for immediate alerts

## Quick Example

Sending a notification from backend code:

```python
from apps.notifications.models import Notification

Notification.objects.create(
    user=user,
    type="CUSTOM_EVENT",
    data={"message": "Something happened!"}
)
```
