## What's Included

The built-in authentication system provides:

- ✅ Email/password authentication
- ✅ Social login (Google, Facebook)
- ✅ Two-factor authentication (2FA)
- ✅ Email verification
- ✅ Password reset
- ✅ Role-based access control (Owner, Admin, Member)
- ✅ User profile management

## Key Files

When customizing authentication, you'll work with:

| Location | Purpose |
|----------|---------|
| `packages/backend/apps/users/` | Backend user models, serializers, views |
| `packages/webapp/src/modules/auth/` | Frontend auth components and hooks |
| `packages/webapp-libs/webapp-tenants/` | Multi-tenant role management |
