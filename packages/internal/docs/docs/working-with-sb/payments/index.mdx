## Configuration

Stripe is configured via environment variables:

| Variable | Description |
|----------|-------------|
| `STRIPE_LIVE_SECRET_KEY` | Stripe API secret key (production) |
| `STRIPE_TEST_SECRET_KEY` | Stripe API secret key (testing) |
| `STRIPE_LIVE_MODE` | Set to `True` for production |
| `DJSTRIPE_WEBHOOK_SECRET` | Webhook signing secret |

:::info Local Development
In development, use Stripe test mode keys. Use the [Stripe CLI](https://stripe.com/docs/stripe-cli) to forward webhooks to your local environment.
:::
