The dynamic translations system allows you to manage all user-facing text through the Django Admin panel. Translation changes are published to a CDN and loaded by the frontend at runtime—no redeployment needed.

## Quick Reference

### Message Definition

```tsx
import { FormattedMessage } from 'react-intl';

<FormattedMessage 
  defaultMessage="Welcome back" 
  id="Dashboard / Welcome message" 
/>
```

### Syncing Keys

Translation keys are automatically synced on backend startup when you run `pnpm saas up`. 
You can also manually sync:

```bash
cd packages/webapp
pnpm translations:extract-and-sync
```

This uses docker exec to run the sync command directly in the backend container—no authentication required.

### Publishing (Admin Panel)

1. Admin → Translations → Locales
2. Select locale
3. Click **Publish Translations** action

### AI Translation (with OpenAI)

1. Admin → Translations → Locales
2. Click **🤖 AI Translate**
3. Select source → target locales
4. Track progress in AI Translation Jobs
